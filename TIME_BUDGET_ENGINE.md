# Time-Budget Engine

## Priority Tiers

1. Safety preparation + active primary skill
2. Power work + main strength exercise
3. High-priority compounds
4. Weekly-volume completion
5. Arms, calves, grip, isolation, optional skills

## Trimming Rules

When predicted session > 60 minutes:

1. Improve pairings and equipment order
2. Move lowest-tier accessory set to compatible day
3. Reduce optional set only if weekly minimum remains satisfied
4. Never shorten power or heavy strength rest
5. Never delete primary skill
6. Display what changed and why

## Estimation Inputs

- Warm-up time
- Working sets × rep/hold duration
- Planned rest
- Exercise transitions
- Superset transitions
- Historical actual rest
- Equipment availability
