# Volume Engine

## Categories

- **Direct primary set**: 1.0 effective set
- **Secondary contribution**: 0.5 effective set by default
- **Technique-only set**: 0 hypertrophy sets
- **Power set**: 0 hypertrophy sets
- **Static hold**: tracked as static exposure seconds, not auto-counted as dynamic sets
- **Painful/poor-form set**: 0 volume

## Muscle Groups Tracked

Chest, lats, upper back, quadriceps, hamstrings, glutes, anterior/lateral/rear deltoids, biceps, triceps, brachialis, forearms, grip, calves, trunk, hip flexors, scapular stabilizers.

## Skill Volume

Hard weighted skill sets count as full sets. Technique practice counts as exposure only. Static holds count as static exposure.
