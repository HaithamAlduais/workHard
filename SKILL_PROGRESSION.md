# Skill Progression

## Rules

1. Change only one primary difficulty variable at a time
2. Require repeatability (two of three exposures normally, three for high-risk)
3. Apply 20% quality-drop stop rule
4. Skills never progress from a single lucky attempt
5. Manual achievements are flagged as self-reported
6. Failure is not the target — stop before breakdown or pain

## Static Holds

Progression priority:

1. Improve quality
2. Reach target duration
3. Repeat across sessions
4. Reduce assistance / advance leverage
5. Add external load only after mastery

## Dynamic Skills

1. Add reps within current node
2. Then reduce assistance or select harder node
3. For weighted: add load at top of range, reset reps

## Unlock Verification Levels

- `self-reported`
- `video-verified`
- `coach-verified`
- `system-verified`

Expert nodes require `video-verified` or `coach-verified`.
