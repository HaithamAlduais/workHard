# Progression Engine

## Strength Progression

For 3–5 or 4–6 rep ranges:

1. If pain reported or deload requested → HOLD_FOR_SAFETY / reduce load
2. If two or more sets miss minimum range, involuntary failure, or significant pain → reduce load ~5%
3. If all sets reach top of range with good form and 1+ RIR → add load
   - Upper body: +1.25–2.5 kg
   - Lower body: +2.5–5 kg
4. If all sets meet minimum range but not top → add one total repetition
5. Otherwise → maintain load

## Hypertrophy Progression

Double progression:

1. Keep load, add one total rep
2. When all sets reach top of range with good form and RIR → add load (smallest increment)
3. Reset reps to bottom of range

## Power Quality

Stop when velocity/quality drops (`slower`). Preserve speed and quality.

## Static Holds

1. Improve quality
2. Reach target duration
3. Repeat across sessions
4. Reduce assistance / advance leverage
5. Add external load only after mastery

## Dynamic Skills

1. Add reps within current node
2. Then reduce assistance or select harder node
3. Never add reps and leverage together
